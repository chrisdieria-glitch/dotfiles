# Dotfiles Project

Dotfiles personales para Arch Linux con Hyprland (Wayland).

## Componentes
- **Hyprland**: Window manager (hyprland, hypridle, hyprlock, hyprpaper) + scripts
- **Kitty**: Terminal emulator
- **Waybar**: Status bar
- **Wofi**: Application launcher
- **Neovim**: Editor (LazyVim)

## Instrucciones

1. Respetar la estructura modular existente
2. Usar sintaxis válida de Hyprland (nunca inventar opciones)
3. Preservar configs existentes; solo modificar si se pide explícitamente
4. Priorizar corrección sobre completitud

## Reporte de iteraciones

**IMPORTANTE**: Después de cada iteración en el proyecto, crear o actualizar el archivo `docs.md` con un reporte que incluya:
- Fecha y hora
- Qué se hizo (archivos modificados, cambios realizados)
- Problemas encontrados (si los hubo)
- Estado actual del proyecto

Este archivo sirve como historial para futuras sesiones.
